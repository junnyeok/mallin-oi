새 글 쓰기

제목/요약/태그/고정 설정 후 등록하면, posts.json용 JSON과 posts/pXXX.html 템플릿이 생성돼.
⭐️오른쪽 코드를 박준혁한테 복붙해서 보내주면 새글을 대신 올려줄 거야.

팁) <h2>, <p>, <ul> 같은 HTML로 써도 되고, 그냥 텍스트로 써도 돼.

#은 붙여도 자동으로 제거돼. 중복 태그는 알아서 정리함.

출력

posts.json에 추가할 JSON



                  
posts/pXXX.html 템플릿
